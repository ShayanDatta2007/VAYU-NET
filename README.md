
# 🌍 VAYU-NET
### AI-Powered Urban Air Quality Intelligence for Smart City Intervention

VAYU-NET is a modern, AI-inspired web application that helps monitor, visualize, and understand urban air quality in real time. It combines live environmental data, interactive analytics, pollution forecasting, and health recommendations into a single user-friendly dashboard. The platform is designed for citizens, researchers, students, and smart city authorities to make data-driven decisions for a healthier environment.

🌐 **Live Demo:** https://shayandatta2007.github.io/VAYU-NET/

---

## 📌 Features

- 🌍 Real-Time Air Quality Monitoring
- 📊 Interactive AQI Dashboard
- 📈 Historical Air Quality Trends
- 🤖 AI-Based Pollution Prediction
- ❤️ Personalized Health Recommendations
- 🗺 Interactive Pollution Map
- 🌤 Weather Information Integration
- 📱 Fully Responsive Design
- 🌙 Modern Glassmorphism UI
- ⚡ Smooth Animations and Transitions

## 🚀 Getting Started

### Clone the repository

```bash
git clone https://github.com/shayandatta2007/VAYU-NET.git
```

### Navigate to the project

```bash
cd VAYU-NET
```

### Run the project

Simply open the `index.html` file in your browser.

Or, if you use VS Code, install the **Live Server** extension and click **Go Live**.
